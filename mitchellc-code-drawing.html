Process

My inspiration piece was Rose Horizontal by Bridget Riley. The original piece is an abstract screen print of varying pastel and musted colors. To recreate this I used an HTML color codes website to color pick from the original piece to get as close as possible. After getting all 25 colors I started on the stripes. Each stripe was the same so I made one stripe using a div then copied that and changed the color, z-index, and margin from the top. I also changed the background color of the page to match the surrounding frame color of the original print.

Original Piece: https://www.artsy.net/artwork/bridget-riley-rose-horizontal-2
